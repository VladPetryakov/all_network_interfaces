# All Network Interfaces

Проект представляет собой набор практических заданий и примеров работы с сетевыми интерфейсами и технологиями.

## Основные задачи проекта:
1. Изучение и работа с сетевыми интерфейсами
2. Применение сетевых протоколов  
3. Практика работы с Docker и контейнеризацией
4. Реализация RPC (Remote Procedure Call)
5. Работа с gRPC для удаленных вызовов процедур
6. Конфигурация и управление сетевыми сервисами

## Технологии:
- Docker
- gRPC
- Selenide
- RPC
- Сетевые протоколы
- Контейнеризация

Проект служит учебным пособием для освоения современных сетевых технологий и инструментов разработки распределенных систем.
