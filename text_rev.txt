Проект "all_network_interfaces" - это репозиторий, содержащий различные задания и задачи.

Обзор структуры проекта:

1. task_1 - Первое задание (конкретное содержание неизвестно без просмотра файлов)
2. task_4 - Четвертое задание
3. task_5 - Пятое задание
4. task_6 - Шестое задание
5. task_7_remote_procedure_call - Задание по удалённому вызову процедур
6. task_8_grpc_calculator - GRPC калькулятор
7. task_9_docker - Задание по работе с Docker
8. task_10_currency-exchange - Обмен валют
9. task_11_selenid - Задание по работе с Selenide
10. task_12_docker_selenid - Docker с Selenide

Также в корне проекта есть несколько текстовых файлов:
- 229.txt - предположительно содержит информацию или код
- commit.txt - пустой файл (возможно для коммитов)
- new.txt - содержит небольшой текст (34 байта)
- new_text_file228.txt - пустой файл
- test_text - небольшой текстовый файл (33 байта)
- text.txt - пустой файл

Проект представляет собой сборник различных практических заданий, связанных с сетевыми интерфейсами, удалёнными вызовами, Docker, тестированием и другими технологиями.
